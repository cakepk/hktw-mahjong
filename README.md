# 🀄 港式台牌小帳本 (HK-Taiwan Mahjong Ledger)

一款專為「港式台灣牌」玩家設計的輕量級計番與記帳工具。無需下載，開啟網頁即可即時記錄牌局狀況，支援特殊的「倍半」與「拉莊」計算規則。

## ✨ 特色功能 (Highlights)

- **自動計番與拉莊**：內建詳細番數表，自動處理「莊家倍半 (1.5x)」與「反擊/平家 (0.5x)」的進位與捨去邏輯。
- **即時快捷付錢**：針對「追、暗槓、一台草/花、詐胡」提供快速按鈕，不影響莊家連莊也能即時更新餘額。
- **智能流水帳**：完整紀錄每一局的輸贏明細，支援滾動查看 100 條以上紀錄，並提供「復原上一手」功能。
- **手動靈活性**：支援手動輸入番數、手動平帳（例如飯錢或借錢紀錄），靈活應對各種真實牌局狀況。
- **隱私安全**：所有數據僅儲存於瀏覽器本地 (`localStorage`)，無須聯網，無須資料庫，數據絕不上傳雲端。

## 🚀 快速開始 (Quick Start)

1. **GitHub Pages 部署**：將此 Repo clone 或上傳到 GitHub，開啟 Pages 功能即可。
2. **本地執行**：直接開啟 `index.html` 即可使用。

## 🛠️ 技術栈 (Tech Stack)
- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS (Styling)
- **Icons**: FontAwesome & [FluffyStuff/riichi-mahjong-tiles](https://github.com/FluffyStuff/riichi-mahjong-tiles) (CC0)
