# 🀄 港式台灣麻雀記帳本 (Administrative Ledger System)

一款專為「港式台灣牌」玩家設計的高端專業計番與數據分析工具。採用「ADMINISTRATIVE LEDGER SYSTEM」專業設計，支援複雜的「倍半」、「拉莊」及「踢半」計算規則，並提供深度的對局戰後分析。

## ✨ 特色功能 (Highlights)

- **專業計番與拉莊系統**：內建詳細番數表，精準處理「莊家倍半 (1.5x)」與「平家踢半 (0.5x)」的進位與扣減邏輯。
- **命運曲線 (Fortune Line)**：動態 SVG 折線圖即時顯示四家餘額走勢，一目了然誰在連莊、誰在止蝕。
- **戰績焦點 (Session Highlights)**：自動計算 8 項關鍵指標（勝率、自摸率、最高番數、平均番數、連莊紀錄、單筆最高斬獲等）。
- **一鍵專業截圖 (Export Summary)**：內建「參謀部戰績摘要」模版，一鍵輸出高品質 PNG 圖片，方便分享對局結果。
- **即時付錢與手動平帳**：提供「暗槓、詐胡、一台草」等即時結算功能，並支援手動平帳 (Manual Adjustment) 處理非對局性金額流入/流出。
- **隱私與安全**：數據 100% 儲存於本地 (`localStorage`)，無須網路，無須註冊，數據絕不離身。

## 🚀 快速開始 (Quick Start)

1. **GitHub Pages 部署**：直接將 Repo 上傳至 GitHub 並啟用 Pages。
2. **本地開啟**：直接下載並執行 `index.html`。
3. **建議環境**：推薦使用 Chrome / Safari / Edge 等現代瀏覽器以獲得最佳視覺體驗。

## 🛠️ 技術庫 (Tech Stack)
- **Engine**: Vanilla JavaScript (ES6+)
- **Styling**: Tailwind CSS (PostCSS) & Heroicons
- **Export**: [html2canvas](https://html2canvas.hertzen.com/)
- **Icons**: FontAwesome & [riichi-mahjong-tiles](https://github.com/FluffyStuff/riichi-mahjong-tiles)
